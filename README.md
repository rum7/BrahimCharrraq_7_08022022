# INSTALLATION
## BACK END
Lancer le terminal dans le dossier backend et effectuer ces commandes :
```
npm install
node server
```

## FRONT END
Lancer le terminal dans le dossier frontend et effectuer ces commandes :
```
npm install
npm start
```<br><br><br>
